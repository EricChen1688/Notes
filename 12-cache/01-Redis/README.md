# Redis

## **RE**mote **DI**ctionary **S**ervice

官网介绍:https://redis.io/topics/introduction
中文网站:http://www.redis.cn

硬件层面有 CPU 的缓存;浏览器也有缓存;手机的应用也有缓存。我们把数据缓存
起来的原因就是从原始位置取数据的代价太大了，放在一个临时位置存储起来，取回就 可以快一些。

## Redis 的特性:

- 1)更丰富的数据类型 
- 2)进程内与跨进程;单机与分布式 
- 3)功能丰富:持久化机制、过期策略 
- 4)支持多种编程语言 
- 5)高可用，集群
