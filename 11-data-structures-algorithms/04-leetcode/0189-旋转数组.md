# [0189-旋转数组](https://leetcode-cn.com/problems/rotate-array/)



## 解法1.临时数组

```java
class Solution {
    public void rotate(int[] nums, int k) {
        int n = nums.length;
        int temp[] = new int[n];
        for(int i= 0; i < n; i++) {
            temp[(i + k) % n] = nums[i];
        }
        System.arraycopy(temp, 0, nums, 0 ,n);
    }
}
```

**复杂度分析**

- 时间复杂度： O(n)，其中 n*n* 为数组的长度。
- 空间复杂度： O(n)。
