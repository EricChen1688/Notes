# 寻找数组的中心索引

https://leetcode-cn.com/problems/find-pivot-index/

## 

- 求和
- 遍历
- 左边和 = 总和 - 左边和 - 当前值

```java
class Solution {
    public int pivotIndex(int[] nums) {
        int sum = 0, leftsum = 0;
        for (int x: nums) sum += x;
        for (int i = 0; i < nums.length; ++i) {
            if (leftsum == sum - leftsum - nums[i]) return i;
            leftsum += nums[i];
        }
        return -1;
    }
}
```
