# Redis 字符串两种存放方式

- 超过 44 字节,使用 raw

## 为什么分界线是 44 字节?

Redis 对象头结构

```java
typedef struct redisObject {
     type;   		  //4bits
     encoding;    //4bits
     lru:LRU_BITS;// 24bits
     refcount;    //4 bits
     *ptr;			  // 64 bit
} robj;
```
