# 分布式事务的理论模型

分布式事务问题也叫分布式数据一致性问题,简单来说就是如何在分布式场景中保证多个节点的数据一致性

分布式事务产生的核心原因是因为存储资源的分布性,比如多个数据库,或者 MySQL和 Redis 两种不同设备的数据一致性等等

我们应该从设计层面尽量避免分布式事务问题,因为任何一种解决方案都会增加系统的复杂度
