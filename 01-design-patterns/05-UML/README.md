# UML

## UML定义

统一建模语言(Unifed Modeling Language,UML) 是一种对面向对象系统的产品进行**说明、可视化和编制文档的标准语言**,是非专利的第三代建模和规约语言

- 是一种面向对象设计的建模工具
- 是在开发阶段说明,可视化,构建和熟悉恶一个面向对象软件密集系统的制品的开放方法
- 独立于任何程序设计语言

## 应用场景

**UML 最佳的应用是工程实践面对大规模,复杂系统进行建模方法,特别是软件架构层次,已经被验证有效**
